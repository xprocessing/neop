## 🔧 配置说明

### README.md 包含的配置说明

#### 1. 后端配置
```yaml
# 数据库配置
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/neop
    username: root
    password: your_password

# Redis 配置
spring:
  redis:
    host: localhost
    port: 6379

# 微信配置
wechat:
  appid: your_wechat_appid
  secret: your_wechat_secret
```

#### 2. 前端配置
```javascript
// API 地址配置
const BASE_URL = 'http://localhost:8080/neop'
```

#### 3. Nginx 配置
```nginx
server {
    listen 80;
    server_name admin.neop.com;
    root /usr/share/nginx/html/neop-admin;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    location /prod-api/ {
        proxy_pass http://localhost:8080/neop/;
    }
}
```

## 🚀 部署说明

### 1. 后端部署
```bash
# 打包
mvn clean package -DskipTests

# 启动（使用 systemd）
systemctl start neop
systemctl status neop
```

### 2. 前端部署
```bash
# 打包
npm run build

# 上传到服务器
scp -r dist/* root@your_server:/usr/share/nginx/html/neop-admin
```
