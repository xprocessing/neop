## 三、阶段7：6个定时任务（文档第十二章）

### 3.1 定时任务清单

**文件**：`ScheduledTasks.java`

| 任务 | 频率 | 描述 |
|------|------|------|
| `taskExpireScan()` | 每30分钟 | 扫描过期任务，自动关闭（状态1→5） |
| `dailyStatistics()` | 每日凌晨0点 | 统计昨日数据，插入data_statistics表 |
| `payOrderTimeout()` | 每5分钟 | 关闭超时未支付订单（超过30分钟） |
| `memberExpireScan()` | 每日凌晨1点 | 扫描过期会员，更新状态为0 |
| `transferRetry()` | 每10分钟 | 重试失败的企业付款（最多5次） |
| `autoConfirmReceipt()` | 每日凌晨2点 | 自动确认收货（发货后7天） |

### 3.2 关键实现

**任务过期扫描**：
```java
@Scheduled(cron = "0 */30 * * * *")
public void taskExpireScan() {
    List<TaskInfo> expireTasks = taskInfoMapper.selectExpireTasks();
    for (TaskInfo task : expireTasks) {
        task.setStatus(5); // 已过期
        taskInfoMapper.updateById(task);
    }
}
```

**支付订单超时关闭**：
```java
@Scheduled(cron = "0 */5 * * * *")
public void payOrderTimeout() {
    LocalDateTime timeout = LocalDateTime.now().minusMinutes(30);
    List<PayOrder> timeoutOrders = payOrderMapper.selectTimeoutOrders(timeout);
    for (PayOrder order : timeoutOrders) {
        // 1. 关闭微信订单（调用微信API）
        // 2. 更新订单状态为已取消
        // 3. 恢复库存
    }
}
```

**企业付款重试**：
```java
@Scheduled(cron = "0 */10 * * * *")
public void transferRetry() {
    List<TaskPayLog> failedList = taskPayLogMapper.selectRetryList(5);
    for (TaskPayLog log : failedList) {
        wechatPayService.retryTransfer(log.getId());
    }
}
```
