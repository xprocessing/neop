# 2026-06-12 创建用户端Web页面

## 操作内容
在 `neop-web/` 目录下创建了用户端 Web 页面，纯 HTML/CSS/JS，无任何框架。

## 页面结构

```
neop-web/
├── index.html        # 首页（概览数据、最新任务/商品）
├── user.html         # 个人中心（登录/注册、积分、签到、邀请）
├── task.html         # 任务大厅（任务列表、领取、提交、我的任务）
├── product.html      # 积分商城（商品列表、分类筛选、搜索）
├── css/
│   └── style.css     # 全局样式（响应式，适配 PC 和手机）
└── js/
    └── common.js     # 公共工具（API 封装、JWT 认证、分页、Toast）
```

## 技术特点
- 纯 HTML/CSS/JS，零依赖，无框架
- CSS Grid + Flexbox 响应式布局，适配 PC、平板、手机
- 后端 API 对接（`http://localhost:8080`）
- JWT Token 存储在 localStorage
- PC/Mobile 双端适配，含移动端汉堡菜单

## 运行状态
- 服务端口：`3001`
- 访问地址：http://localhost:3001
- 启动命令：`python3 -m http.server 3001`

## 注意事项
- 任务列表、商品列表等公开接口无需登录即可访问
- 领取任务、我的任务、积分查询等需要登录
- 登录接口 `/api/auth/login` 可能因 JWT 拦截器限制，需确认后端排除该路径
- 后端已有 CORS 全放通配置，跨域无问题
