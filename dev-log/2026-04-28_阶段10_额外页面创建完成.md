## 🎨 页面功能说明

### UniApp移动端页面

#### 1. 登录页（login.vue）
- **功能**：手机号+验证码登录
- **特性**：
  - 手机号格式验证
  - 验证码倒计时
  - 微信登录入口
  - 用户协议和隐私政策链接

#### 2. 注册页（register.vue）
- **功能**：手机号注册+邀请码（可选）
- **特性**：
  - 三步验证：手机号、验证码、密码
  - 密码显示/隐藏切换
  - 邀请码选填
  - 协议勾选验证

#### 3. 绑定手机页（bind-phone.vue）
- **功能**：绑定/更换手机号码
- **特性**：
  - 手机号验证
  - 验证码验证
  - 绑定成功提示

#### 4. 忘记密码页（forgot-password.vue）
- **功能**：通过手机号重置密码
- **特性**：
  - 三步流程：验证手机 → 重置密码 → 完成
  - 密码确认验证
  - 重置成功跳转登录

#### 5. 确认订单页（checkout.vue）
- **功能**：订单确认和提交
- **特性**：
  - 收货地址选择
  - 商品清单展示
  - 支付方式选择（微信、支付宝、积分）
  - 订单备注
  - 价格明细（商品总价、运费、优惠、实付金额）

#### 6. 订单详情页（order-detail.vue）
- **功能**：查看订单详细信息
- **特性**：
  - 订单状态展示（待支付、已支付、已发货、已完成、已取消）
  - 收货信息
  - 商品信息
  - 订单信息（编号、时间、支付方式）
  - 底部操作栏（取消、支付、确认收货、评价、删除）

#### 7. 支付结果页（payment-result.vue）
- **功能**：展示支付结果
- **特性**：
  - 支付成功/失败状态
  - 订单信息展示
  - 推荐商品
  - 操作按钮（查看订单、继续购物、重新支付、联系客服）

#### 8. 搜索页（search/index.vue）
- **功能**：全局搜索商品/任务
- **特性**：
  - 搜索历史（保存、清空）
  - 热门搜索
  - 搜索建议
  - 搜索结果（商品、任务两个标签页）

#### 9. 消息通知页（message/index.vue）
- **功能**：查看系统通知、订单通知、活动通知
- **特性**：
  - 消息分类（全部、系统、订单、活动）
  - 未读消息红点
  - 全部已读功能
  - 点击跳转到相关页面

#### 10. 设置页（settings.vue）
- **功能**：应用设置
- **特性**：
  - 账户安全（手机号、修改密码、生物识别）
  - 通知设置（系统、订单、活动）
  - 通用设置（清除缓存、字体大小、深色模式）
  - 关于（关于我们、帮助中心、当前版本）
  - 退出登录

#### 11. 关于我们页（about/index.vue）
- **功能**：展示公司信息
- **特性**：
  - 公司名称、地址、电话、邮箱、官网
  - 功能链接（用户协议、隐私政策、帮助中心）
  - 分享应用
  - 应用评分

#### 12. 帮助中心页（help/index.vue）
- **功能**：常见问题解答
- **特性**：
  - 常见问题FAQ（可展开/收起）
  - 问题搜索
  - 联系客服（电话、在线客服）
  - 意见反馈
  - 用户协议、隐私政策链接

## ⚠️ 需要人工介入的事项

### 高优先级
1. **配置微信支付证书**：上传p12证书至 `/data/neop/cert/`
2. **配置小程序AppID**：编辑 `neop-uniapp/manifest.json`
3. **配置后端API地址**：编辑 `neop-uniapp/utils/request.js` 和 `neop-admin-web/src/utils/request.js`
4. **创建MySQL数据库**：执行Flyway脚本创建28张表

### 中优先级
5. **配置Redis连接**：编辑 `neop-backend/src/main/resources/application.yml`
6. **配置文件上传**：选择COS/OSS/本地存储并配置
7. **安装依赖并启动**：
   ```bash
   # 后端
   cd neop-backend && mvn spring-boot:run
   
   # Vue3后台
   cd neop-admin-web && npm install && npm run dev
   
   # UniApp
   # 使用HBuilderX打开neop-uniapp目录
   ```
